--------------------------------------------------------------------------------
WINDOWS USERS
--------------------------------------------------------------------------------
For using the BASS library, place a copy of bass.dll into this folder. 
Download from https://www.un4seen.com/download.php?bass24
Note that commercial application of BASS is not free.

For using the free UOS library, place a copy of the following dlls into this
folder
- LibPortaudio-32.dll
- LibSndFile-32.dll
- LibMpg123-32.dll  

Download precompiled binaries for Windows from
- https://github.com/fredvs/uos/tree/master/examples/lib/Windows/32bit
- https://www.mpg123.de/download.shtml
- https://github.com/bastibe/libsndfile-binaries

In the cases where the names are different rename the files to match above
specification.

Be careful to select the version with the correct bitness.
